# Server-Side-Project

Full stack e-commerce book store web application made as a part of Server-side Scripting Frameworks course at Metropolia University of Applied Sciences Software Engineering field.

## App stack
- Node.js
- TypeScript
- MongoDB
- GraphQL
- REST
- Bootstrap
- passport.js
- jsonwebtoken
- stripe
- MVC architecture
